💬 Telegram

Como configurar Telegram:

  1. Habla con @BotFather en Telegram
  2. Usa /newbot para crear un bot
  3. Copia el token que te da
  4. Cada usuario debe enviar /start a tu bot
  5. Usa @userinfobot para obtener Chat IDs
